
# **Extraction 2D Platformer Java game + Game Engine**

The purpose of this project is for me to learn and use java as a programming lanuage and use Object orientated programming. The Project is a 2D Platformer game engine with a Extraction type game built inside of it.

---

## Features

- **No dependentcies** — This game was built using the standard java libraries.
- **Animations and sprites** — Sprites sourced from itch.io and animations constructed in game and changed dynamically.
- **Collision Detection** — Collision detection system.
- **Game map** — Map loaded from txt file. Background is a paralax scrolling background.

## Architecture

I have tried to keep the Game engine and Game seperate in the project. Therefore you will find the game engine structure within the Engine package and the game structure inside of the Game package. There is also a Main package that just containes the Main.java
